Cosmic Cafe — in-game background photos (Alien Jump)

1. Copy your images from your PC into this folder (e.g. from Downloads\cosmic photos).
2. Supported: .jpg, .jpeg, .png, .webp
3. Edit manifest.json and list each filename, for example:
   ["bar1.jpg", "bar2.png"]

The game loads these as a soft background layer behind the stars. If manifest.json is empty [], only the default gradient + stars are shown.
